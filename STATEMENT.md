## Problem Statement
College students often struggle to find books that match their interests, it becomes time consuming
A system is needed that can help students search books by title, author, or genre, and provide recommendations based on similar genres.

## Scope of the Project
- Helps students and readers quickly find books of interest  
- Provides recommendations based on genre similarity  
- Can be expanded later with database integration or AI-based suggestions  

## Target Users
- College students  
- Library users  
- Beginner readers seeking book recommendations  

## High-Level Features
1. Search books by *title, author, or genre*  
2. Display all genres if no search input is given by clicking search button

4. Alerts for invalid selections or empty searches
